# Dittocade Components

Shared UI components for the Dittocade.

## Building

You'll need `git`, `node 18+` and `npm` to get started.

```sh
# Cloning the repository
git clone https://github.com/dittocade/comps
cd ./comps

# Installing dependencies
npm ci
```
